# Анализ маркетинговой компании
## Описание исследования
**Заказчик:** Procrastinate Pro+ — развлекательное приложение.

**Цель исследования:** Несмотря на огромные вложения в рекламу, последние несколько месяцев компания терпит убытки. Необходимо разобраться в причинах и помочь компании выйти в плюс.

**Входные данные:**  Данные о пользователях, привлечённых с 1 мая по 27 октября 2019 года:
* лог сервера с данными об их посещениях,
* выгрузка их покупок за этот период,
* рекламные расходы.

**Задачи исследования**: Необходимо ответить на вопросы:
1. откуда приходят пользователи и какими устройствами они пользуются,
2. сколько стоит привлечение пользователей из различных рекламных каналов;
3. сколько денег приносит каждый клиент,
4. когда расходы на привлечение клиента окупаются,
5. какие факторы мешают привлечению клиентов.
